### 项目说明
这是一个小的vue项目
vue2 + axios + elementui
其中数据接口采用天行数据平台公开的接口（https://www.tianapi.com/）
在线地址：www.acaia.top
如果你有需要，可以随意copy，甚至拿去当课堂作业。
后续有时间的话我会继续维护或更新新功能。
